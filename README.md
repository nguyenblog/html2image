# HTML to PDF Node.js

Chuyển đổi file HTML sang PDF sử dụng Node.js và Puppeteer.

## Cài đặt

```bash
npm install
```

## Sử dụng

```bash
node index.js <input.html> <output.pdf>
```

Ví dụ:
```bash
node index.js example.html output.pdf
```

## Ghi chú
- Bạn cần Node.js >= 14
- Puppeteer sẽ tự động tải Chromium khi cài đặt
